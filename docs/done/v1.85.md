# v1.85 完成记录

## 已完成
- 新增数据表：`sys_box_circuit`，支持一个配电箱多条回路。
- 新增后端模块：
  - `BoxCircuit` 实体
  - `BoxCircuitMapper`
  - `IBoxCircuitService` / `BoxCircuitServiceImpl`
  - `BoxCircuitController`（`/box-circuit`）
- 新增接口能力：
  - `GET /box-circuit/{boxId}` 查询当前配电箱全部回路
