# v1.28 完成记录

## 本次改动
1. 在配电箱详情页新增“编辑基础信息”入口按钮。
2. 新增详情页内基础信息编辑弹窗。
3. 明确将 `boxId`（台账号）设为不可编辑（disabled），并在保存时强制使用原值，防止被改写。

## 实现说明
- 详情页工具栏新增 `编辑基础信息` 按钮。
- 打开弹窗时回填当前配电箱基础信息：`boxId/station/area/boxAddress/size`。
- 保存时调用既有 `/box/save`：
  - 可编辑字段：`station/area/boxAddress/size`
  - 不可编辑字段：`boxId`（始终取当前详情原值）
- 保存成功后刷新详情数据，确保页面展示与数据库一致。

## 验证建议
1. 进入某配电箱详情页，点击“编辑基础信息”。
2. 确认“台账号”输入框不可编辑。
3. 修改车间/工区/地址/规格并保存。
4. 页面刷新后显示新值；台账号保持不变。
