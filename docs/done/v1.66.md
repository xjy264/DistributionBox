# v1.66 完成记录

## 根因
图片由浏览器直接请求，不会自动携带 axios 的 token 请求头。恢复鉴权后，图片接口被拦截返回 401，导致全站图片 `Failed`。

## 修复
在 `SecurityConfig` 中仅放行图片读取 GET 接口：
- `/files/**`
- `/files/preview/**`

保留其它业务接口鉴权不变。
