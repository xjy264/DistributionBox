# v1.67 完成记录

## 根因
前端角色与菜单页面调用了错误接口前缀（`/roles`、`/menus`），而后端实际路由为 `/role`、`/menu`，导致请求异常并触发前端未授权处理流程。

## 修复
- `Role.vue` 接口统一改为 `/role`：
  - `/role/page`
  - `/role`
  - `/role/{id}`
  - `/role/checks/{id}`
  - `/role/roleMenu/{id}`
  - 菜单ID查询改为 `/menu/ids`
- `Menu.vue` 接口统一改为 `/menu`：
  - `/menu`
  - `/menu/{id}`

## 结果
登录后进入“角色/菜单”页面不再误判鉴权失败跳转登录页。
