# v1.77 完成记录

## 根因
前端 axios 401 拦截器对任意 401 都会触发 logout + 跳转登录；生产环境图片请求（`/files/**`）返回 401 时，登录后进入页面会被误判为会话失效，造成回跳 `/login`。

## 修复
- 调整 `frontend/src/api/http.ts`：
  - 401 拦截中增加忽略路径：
    - `/files/`
    - `/api/files/`
    - 既有的 `/dict/`、`/location/`
  - 仅在非忽略业务接口 401 时才执行 logout + 跳转登录。

## 结果
登录后不再因图片 401 触发误登出，避免“刚登录就跳回 login”问题。
