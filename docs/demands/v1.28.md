# v1.28 需求文档

## 需求背景
当前配电箱编辑入口主要在列表页。用户希望在“配电箱详情”页直接完成基础信息编辑，减少来回跳转。

## 目标
1. 在配电箱详情页提供“编辑基础信息”入口。
2. 编辑时明确限制：`台账号(boxId)` 为只读不可修改。
3. 保存后刷新详情页数据，保持与现有保存接口一致。
4. 不影响现有图片上传/回显/落库链路。

## 范围
- 前端：`frontend/src/views/BoxDetail.vue`（必要时复用现有 `EntityForm`）
- 后端：原则上不新增接口，沿用 `/box/save`。

## 验收标准
1. 详情页可打开“编辑基础信息”弹窗/区域。
2. `boxId` 输入不可编辑（disabled 或 readonly），保存时不被修改。
3. 其他允许字段可正常保存并刷新展示。
4. 前端构建通过。
