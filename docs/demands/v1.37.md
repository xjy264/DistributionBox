# v1.37 需求文档

## 目标
基于现有配电箱后端（127.0.0.1:8888）实现微信小程序首版业务能力，跳过 openid 登录，直接复用账号密码登录。

## 范围
1. 小程序项目目录：`/Users/xuejiayao/WeChatProjects/DistributionBox`
2. 页面：登录、配电箱列表、配电箱详情、巡检列表/新增、抢修列表/新增、我的
3. 复用后端接口：`/login`、`/box/*`、`/inspection-item/page`、`/inspection-task/save`、`/repair-item/page`、`/repair-task/save`

## 验收
- 小程序可登录并携带 token 调用后端。
- 可查看配电箱列表和详情。
- 可提交巡检/抢修记录到数据库。
