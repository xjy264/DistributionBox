# v1.40 需求文档

## 目标
将配电箱详情中的“巡检/抢修”语义升级为“维保/检修”，并修复详情页新增按钮点击无响应问题。

## 范围
1. 前端详情页文案改名：巡检明细历史->维保记录，抢修记录->检修记录。
2. 前端详情页接口改用 maintenance/overhaul 语义路径。
3. 修复新增维保/检修弹窗无法打开或保存无响应（增加错误兜底提示）。
4. 后端新增 maintenance/overhaul 路径并兼容 inspection/repair 旧路径。
5. 数据库新增兼容视图迁移，保留旧表不破坏数据。

## 验收
- 点击“新增维保任务/新增检修任务”可打开弹窗。
- 保存后提示成功并刷新列表。
- 新旧接口路径都可访问。
