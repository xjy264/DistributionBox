# v1.26 需求文档（修复配电箱详情基础信息显示异常）

## 背景
当前在“配电箱详细信息”页面中，配电箱基础信息（如配电箱ID、台账号等）存在无法正常显示的问题，影响数据查看与演示。

## 需求
1. 修复配电箱详情页基础字段显示异常。
2. 确保至少以下字段可稳定显示：
   - 配电箱ID（id）
   - 台账号（boxId）
   - 车间（station）
   - 工区（area）
   - 安装地点（boxAddress）
3. 对空值/异常值做兜底显示（如 `-`），避免页面出现空白或错位。
4. 不破坏现有巡检/抢修/元器件等详情页功能。

## 期望结果
- 进入任意配电箱详情页，基础信息区能正确显示对应字段值。
- 页面在路由参数不完整、后端返回慢或部分字段为空时仍有稳定表现。
