# v1.30 需求文档

## 需求
修复配电箱详情页图片上传链路，覆盖系统图与图片1/2/3/4。

## 问题现象
1. 选择图片后不能正确回显。
2. 图片 URL/路径未正确设置。
3. 用户后续会从本地上传图片，系统需稳定存储到服务器并可回显。

## 目标
1. 系统图 + 图片1/2/3/4 上传后立即回显。
2. 上传返回值可兼容并正确回填 URL/路径。
3. 保存后字段落库，重进详情页仍可显示。
4. 前端上传链路保留一套，兼容现有后端上传接口差异。

## 验收标准
1. 在详情页分别上传系统图与图片1~4，均可即时看到预览。
2. 点击保存后数据库字段更新。
3. 重新打开详情页，图片仍正确显示。
